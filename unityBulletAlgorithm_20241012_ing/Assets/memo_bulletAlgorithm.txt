i) 좌표계 coordinate system
: 물체의 '위치'를 지정하기 위한 시스템

	가) 데카르트 좌표계( 직교 좌표계 ) Cartesian coordinate system
		
		기저벡터 들은 서로 원점에서 직각으로 만난다는 것이 가장 큰 특징이다.
		<--스칼라(실수)의 순서쌍으로 표기한다. (x,y)

	나) 극좌표계 Polar coordinate system

		'각도'와 극점(원점)에서의 '거리' 로 표기한다.

좌표계의 의의는 기하학Geometry과 대수학Algebra의 접점이라는 것이다.



ii) 벡터 vector

	'크기' 와 '방향' 을 아우르는 개념
	( 스칼라는 크기만 나타낸다. )

	가) 표기법
		
		수벡터 : 벡터의 대수적 표현

			스칼라(실수)의 순서쌍으로 표기한다. (x,y)
			<--위치랑 표기법이 같다.

			(1,0)
			<--방향 (1,0)
			<--크기 1

		기하벡터 : 벡터의 도형 표현

			화살표로 벡터를 나타낸다.

			화살표가 지향하는 곳이 '방향'이고 
			화살표의 길이가 '크기'이다.
	

	나) 벡터의 연산

		a) 벡터끼리의 덧셈(뺄셈)

			A = (1,0)
			B = (0,1)

			A + B = (1,0)+(0,1) = (1+0,0+1) = (1,1)

			각각의 구성성분끼리 더한다(뺀다)

		b) 벡터의 스칼라곱셈

			A = (1,0)
			S = 2

			A*S = (1,0)*2 = (1*2, 0*2) = (2,0)

		c) 벡터끼리의 곱셈
				
				*) 벡터의 내적
					A dot B = ||A|| ||B|| cosT
					
					벡터끼리의 내적은 결과로 스칼라가 나온다.

				*) 벡터의 외적
					A cross B = ||A|| ||B|| sinT U
					(U는 A 와 B에 모두 수직인 단위벡터)

					벡터끼리의 내적은 결과로 벡터가 나온다.

		d) 벡터의 크기

			자기자신을 내적한 것에 제곱근을 구한다.

		e) 벡터의 정규화
			임의의 벡터를 크기가 1인 단위벡터로 만드는 연산

			벡터의 크기 분의 1을 스칼라 곱셈한다.
			
		단위벡터: 크기가 1인 벡터
